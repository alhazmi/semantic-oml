ssize_t base64_validate_string ( const char *  s)

Validate a BASE64-encoded string.

A properly formed string will have a length which is a multiple of 4 bytes and all characters will be drawn from the BASE64 character set. Note that the empty string is considered to be a valid BASE64 string.

Parameters
sA non-NULL pointer to the BASE64-encoded string.
Returns
If valid the length of the string minus any padding; otherwise -1.
