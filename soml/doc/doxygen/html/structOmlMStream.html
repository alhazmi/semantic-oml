Detailed Description

Definition of a Measurement Stream.

A measurement stream links an MP to an output, defined by a writing function (OmlWriter), passing some or all of the fields into a filter (OmlFilter).

All the samples injected into an MP are received, but through filtering and aggregation, the output rate of the MS might be different (e.g., 1/n samples, or with a time-based periodicity).

See Also
OmlMP, OmlWriter, OmlFilter, omlc_inject
