int validate_name ( const char *  name)

Validate the name of an OML identifier (schema name, schema column, measurement point, application.)

The only valid characters for the name of an identifier are letters [A-Za-z], numbers, and the underscore character. The first character must not be a digit.

Parameters
namethe name to validate.
Returns
If the name is valid, returns 1, otherwise returns 0.
